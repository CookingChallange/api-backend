# Cooking Challeng - Backend

## Tech Stack
 - NestJS
 - Typescript


## Setup

```bash
$ npm i
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
